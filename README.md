# React Nx Workshop Summer 2025

## Installation Instructions

**System Requirements**

* `node > 22`
* `npm > 10`

**Clone and install**

```bash
git clone https://github.com/push-based/ws-nx-summer2025.git

cd ws-nx-summer2025
npm install

# (optional) if the step before didn't work, please try the following
npm install --force
```

**Run the application**

```bash
npm run start
```
